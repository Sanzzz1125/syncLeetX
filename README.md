# SyncLeetX 🚀

Auto-sync your accepted **LeetCode** and **GeeksForGeeks** solutions to GitHub with a modern analytics dashboard, live stats, recent solved tracking, and auto-generated READMEs — directly from a Chrome extension.

---

## ✨ Features

- 🔥 Auto-sync accepted LeetCode + GFG submissions
- 📂 Difficulty-based folder organization
- 📝 Auto-generated README for every problem
- 📊 Live popup analytics dashboard
- 📈 Interactive progress charts
- 🕒 Recent solved tracker (latest 10)
- 🔗 Clickable recent solved problems
- ⚡ Instant sync after Accepted submissions
- 🎨 Modern premium extension UI
- 🌐 Fully frontend — no backend required

# 🚀 Quick Start

```bash
git clone https://github.com/Sanzzz1125/SyncLeetX.git
```

Then:

1. Open `chrome://extensions`
2. Enable Developer Mode
3. Click `Load unpacked`
4. Select the extension folder

For complete setup guide:

👉 See [SETUP.md](./SETUP.md)
